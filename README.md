# Mindtek Cucumber Automation Test

This project is a test automation framework built using Selenium, TestNG, and Cucumber.  
It is designed to automate web application testing using behavior-driven development (BDD) approach.

---

## 🚀 Technologies

- Java
- Selenium WebDriver
- TestNG
- Cucumber (BDD)

---

## 📌 Project Features

- Page Object Model (POM) design pattern
- Cucumber feature files for test scenarios
- Reusable test components
- TestNG for test execution and reporting
- Structured automation framework

---

## 🧪 Test Scenarios

The project includes automated test scenarios such as:

- User login functionality
- Page navigation
- UI element validation
- Functional test cases for web applications

---

## 🧠 What I Did

- Designed and built a test automation framework from scratch
- Implemented Page Object Model for better maintainability
- Wrote automated test scenarios using Cucumber
- Integrated TestNG for running and managing tests
